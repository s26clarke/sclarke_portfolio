Facebook AI Content Posting Bot

Overview

Digital content 'slop' is on the rise and I wanted to see how easy it would be to create an automated bot to post AI generated content to Facebook to see if it would garner any human interaction.

Written in Python, the application is hosted in Google Cloud to post every hour by generating a novel text quote relating to Facebook cliches via Gemini API. The string would then be returned and formatted into a square image with varying backgrounds and posted via the Facebook Graph API.

Skills: Python, Google Gemini API, Facebook Graph API, Image Generation

Results

I only received limited attention on my posts, likely due to the saturation of AI generated content out there already, Further exploration needs to be done regarding other images/videos that can be generated as well as artificially inflating followers/likes to see if the Facebook algorithm would promote the content more.
